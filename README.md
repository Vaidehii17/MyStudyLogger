MyStudyLogger

MyStudyLogger is a simple Python program that helps record daily study activity.
This project was created to practice basic Python concepts and file handling.

Project Purpose
I created this project to understand how user input can be stored in a file
and later retrieved. It is designed as a small utility that runs in the terminal.

Features
- Add study session details
- Store subject name and time studied
- View previous study records
- Simple menu-based interaction

Technologies Used
- Python
- File handling
- Date module

How to Run
1. Download or clone the repository
2. Open terminal in the project folder
3. Run:
   python study_tracker.py
